The [Reclaim Protocol DevTool](https://dev.reclaimprotocol.org) is where you create applications, manage providers, and monitor your integration's health.

## Getting Started

If you're new to Reclaim Protocol, start by [getting an API key](/api-key) to create your first application.

## Provider Management

Providers define how Reclaim Protocol verifies information from a specific website or API. You can:

- [Explore existing providers](https://dev.reclaimprotocol.org/explore) from the 2500+ data sources already available
- [Create custom providers](https://dev.reclaimprotocol.org/my-providers) for websites not yet supported
- Set up [automation testing](/devtool/automation-testing) to monitor your providers
