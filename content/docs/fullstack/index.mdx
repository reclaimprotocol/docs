# What is fullstack integration

In order to use Reclaim Protocol, you need to configure Providers. But when you have usecases like education or employment verification, you will need to verify across hundreds or thousands of websites, meaning you will have to create as many Providers.

For these usecases, we recommend using our fullstack integration, where we have already created and populated all the universities, employers and government issued identity websites so that you don't have to.

**The UI for selecting the correct portal for the verification is managed by Reclaim Protocol.**

# When to use Fullstack integration?
- **Education Verification**
  - Graduation Year, Major, Registered Name, GPA
  - If you are verifying employment for the purpose of deals/discounts, please [contact us](mailto:support@reclaimprotocol.org). We might have a better offering via our partnership with [SheerID](https://sheerid.com)
- **Employment Verification**
  - Designation, Joining date, Compensation
- **Identity Verification**
  - Age, Official Fullname etc.
